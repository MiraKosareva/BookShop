# Бумажный Феникс — книжный интернет-магазин

Дипломный проект. Разработан на PHP-фреймворке Yii2.

## Демо

Опубликован: **[kosarevam2.temp.swtest.ru](http://kosarevam2.temp.swtest.ru/)**

## Стек

- PHP 8.1, Yii2, MySQL 8.0
- Bootstrap 5, jQuery, AJAX
- Font Awesome 6.5

## Возможности

### Пользователь
- Регистрация/авторизация
- Каталог с фильтрацией, сортировкой и поиском
- Карточка товара с галереей фото
- Корзина и избранное (AJAX, без перезагрузки)
- Оформление заказа (имитация оплаты)
- Личный кабинет с аватаром и историей заказов
- Отзывы с рейтингом и модерацией
- Подписка на рассылку, форма связи с поддержкой
- Адаптивный дизайн (ПК, планшет, телефон)

### Админ-панель
- Дашборд со статистикой
- CRUD-управление книгами (загрузка фото)
- Управление заказами и статусами
- Модерация отзывов + графики
- Просмотр сообщений от пользователей
